# TaylorTech Marketplace SaaS v4

Sistema web leve para simular preços de venda na Shopee e no Mercado Livre.

## Recursos
- Simulador de preço para Shopee e Mercado Livre
- Consulta automática de taxa do Mercado Livre via API pública
- Taxa manual de fallback
- Preço psicológico
- Cadastro local de produtos
- Exportar JSON
- Inserir JSON
- Exportar PDF da simulação atual e dos produtos salvos/importados
- Visual TaylorTech em preto, branco e verde
- Logo digitada no próprio site

## Estrutura
- `index.html`
- `css/style.css`
- `js/script.js`

## Publicação no GitHub Pages
1. Suba os arquivos para um repositório
2. Vá em Settings > Pages
3. Escolha `Deploy from branch`
4. Selecione a branch `main` e a pasta `/root`
