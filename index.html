Documentação rápida

Como o sistema trabalha.

  • O PDF exporta a simulação atual e também a lista de produtos salvos, quando existir.
  • O JSON exporta todos os produtos cadastrados no navegador.
  • O botão Inserir JSON importa produtos salvos para continuar editando e exportando.
  • A API do Mercado Livre tenta consultar custos por preço e mostra os listing types retornados.
  • Se a API não responder, o sistema usa a taxa manual definida por você.
